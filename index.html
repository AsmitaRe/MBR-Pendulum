Modelling (or Derivations)

Building a mathematical system that can 'act out' real world changes

The process of derivation 'loads' real world activity into mathematical equations. These mathematical equations then 'act out' the dyanamic changes in the real world. These equation can thus be thought of as a symbol-based 'machine', which can mimic -- act out -- changes in the real world under given setting (intial values).

These symbol-based machines (models) can be used to estimate or predict the changes in the physical world. These model predictions are then be validated/rejected by comparison with real experimental data.

These validated models allow building and exploring many 'what-if' scenarios, without making (costly) changes to the real world.
